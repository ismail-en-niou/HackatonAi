# Web Frontend Documentation

The web frontend is a Next.js application that provides a user interface for interacting with the AI backend.

## Project Structure

The project is structured as follows:

- **`src/app`**: Contains the main application logic, including pages and components.
- **`src/app/api`**: Contains the API routes for the frontend.
- **`src/app/components`**: Contains reusable React components.
- **`src/app/lib`**: Contains utility functions and services.
- **`public`**: Contains static assets, such as images and fonts.

## Key Components

- **`ChatContainer.jsx`**: The main component for the chat interface.
- **`StreamingChatContainer.jsx`**: A component for handling streaming chat responses.
- **`Navbar.jsx`**: The navigation bar component.
- **`AdminNavbar.jsx`**: The navigation bar for the admin panel.
- **`AdminSidebar.jsx`**: The sidebar for the admin panel.
